Simple project to showcase of PIXI functionalities combined with TypeScript.
Partially covered with JEST tests.

Code is well documented, reusable and easily changeable using constants and readonly properties of classes.

Required
Node version: v20.16.0
NPM version: 10.8.1

To run locally type in console:
npm run dev

To build type in console:
npm run build

To run tests type in console:
npm run test
